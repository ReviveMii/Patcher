# ReviveMii Patcher

Patcher for ReviveMii, to change the old Nintendo Server URL's to ReviveMii's Servers

Only Nintendo DS/DSi Games are supported at the time.

The Servers are not ready at the time

# Build it

Run
```
go build
```
to build it.
